# 与江南 skill 的结构对比

本文件记录本轮优化中对照 `jiangnan-perspective` 后发现的差距和补强动作。

## 江南 skill 做得更完整的部分

1. **事实工作流**：江南 skill 明确区分事实问题、框架问题和混合问题，并要求涉及最新状态时先研究。
2. **身份卡**：江南 skill 有“我是谁、我的起点、我现在在做什么”，能让角色输出更稳定。
3. **公开材料证据链**：江南 skill 把公开访谈、媒体报道、版权材料和本地语料分层。
4. **最新动态**：江南 skill 标出调研日期和近期变化，避免过时判断。
5. **智识谱系/价值观/反模式**：不只给写作技巧，也给作者如何看创作、产业和边界。
6. **任务协议细分**：章节改稿、微观改稿、原创顾问、人物声音校准分开处理。

## 狐尾 skill 原短板

1. 过度依赖本地《道诡异仙》文本结构，缺少作者自述。
2. 没有记录公开访谈中的“灵感库”“反套路”“题材初心”“不擅长感情线”等关键方法。
3. 没有外部评价层，导致多重真实、东方克系、IP 二创价值等判断不够有来源。
4. 没有时间线和最新动态提醒。
5. 原创输出模板偏机制，缺少作者级创作方法：先建素材库、爆点、题材底线。

## 本轮补强

- 新增 `02-conversations.md`：作者公开访谈与创作自述。
- 新增 `04-external-views.md`：作品研讨与外部评价。
- 新增 `05-decisions.md`：创作决策和路线变化。
- 新增 `06-timeline.md`：时间线和最新动态提醒。
- 更新 `SKILL.md`：加入事实研究工作流、身份卡、灵感库模型、反套路模型、题材初心模型。
- 更新决策启发式：加入灵感库、卷内爆点、题材底线、扬长避短。

## 仍不如江南 skill 的地方

- 江南 skill 有更细的人物声音矩阵；狐尾 skill 目前不适合做原作人物仿写，出于版权和近似复刻边界，保留为高层机制。
- 江南 skill 的版权/IP 研究更细；狐尾 skill 只做事实问题查证提醒，未展开影视和实体书权利链。
- 狐尾公开长访谈数量较少，作者表达 DNA 仍应以“创作方法”而不是“人格扮演”作为主轴。
